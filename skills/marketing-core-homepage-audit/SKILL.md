# Marketing Core — Homepage Audit

## Audit checklist
1. Hero clarity (what, for who, why now)
2. CTA strength and placement
3. Proof elements (logos, numbers, testimonials)
4. Friction points (forms, navigation, load)
5. Message consistency above the fold

## Output
- P1/P2/P3 issue list
- Fix recommendation per issue
- Expected impact and test plan

## Rule
- Ship top 3 fixes first; avoid full redesign before quick wins.
