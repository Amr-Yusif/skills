# Marketing Core — Positioning

## Core outputs
- ICP statement
- Problem statement
- Unique mechanism
- Proof block
- Objection rebuttals

## Quality checks
- Must be specific (no generic claims)
- Must include measurable proof where possible
- Must map to one target persona at a time

## Fast template
- For [ICP], we solve [pain] by [mechanism], resulting in [outcome], proven by [evidence].
