# Marketing Core — Daily Brief

## Required sections
1. Growth Snapshot (last 24h)
2. Pipeline Health
3. Content/Creative Ops
4. Market Radar
5. Top 3 Actions (owner + deadline)

## Reliability rules
- If sources fail, send degraded brief (never send empty success).
- Include missing-source note and next retry window.
- Include 3 concrete actions even in degraded mode.

## Output style
- Arabic, direct, no fluff.
- Problem -> Solution -> Next Steps.
